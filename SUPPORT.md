# Support

No support is provided for this repository.

The maintainer does not provide:

- DAH access, bearer tokens, sessions, or account recovery;
- help with DAH billing, associations, residents, apartments, chats, reports, or
  operational data;
- troubleshooting for DAH service availability, DAH permissions, DAH UI
  behavior, or DAH production incidents;
- guarantees that endpoints, payloads, or behavior remain compatible with DAH.

For DAH service, account, privacy, billing, or association issues, contact DAH
through its official support channels.

Do not post bearer tokens, session data, personal data, financial data,
association records, exported reports, chat content, or raw API responses in
public issues, pull requests, discussions, screenshots, logs, or examples.

Requests outside the scope of this source code may be closed without response.
